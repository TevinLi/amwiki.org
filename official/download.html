最新版 Zip 包下载地址：

(具体安装方式，请参照文档中心教程篇)


Cli(终端/命令行) 模式下，虽然也可以通过下载 Zip 覆盖安装位置升级，但是更推荐使用 npm install -g amwiki 升级，并且首次安装必须通过此方式
