# Markdown 复选框 (todo list)

可以使用复选框来标记某个项目各项任务的完成情况

## 语法

&#91;x] A  
&#91;&#32;] B  
&#45;&#32;&#91;x] 已选中  
&#45;&#32;&#91;&#32;] 未选中  

## 说明

- 未选中的中括号间一定要有一个空格
- 在 amWiki 中，可以使用 `√ × X x - _` 6个符号来标示已选中
- 在 amWiki 中，复选框外套一层 list 不是必需的，因此可以将复选框显示到一行
- 复选框外套一层 list 不会显示点号和序号（计划中，敬请期待...）

## 效果

[√] A
[ ] B  
[ ] C

- [x] 已选中
- [ ] 未选中
